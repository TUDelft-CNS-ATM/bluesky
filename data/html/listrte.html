LISTRTE: Listrte

The command will show waypoints list of the aircraft route. The list consists of maximum 7 waypoints per page. It will return information such as:

  • Waypoint name
  • Altitude
  • Speed

When the list is longer, the automatic command for the next page will be displayed.

Usage:

LISTRTE acid

Arguments:

Name Type Required Description
acid txt Yes Aircraft id
