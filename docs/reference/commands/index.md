# Stack command reference

BlueSky is controlled through **stack commands**: short text commands typed
into the console, played back from [scenario files](../../user-guide/scenario-files.md),
or issued by plugins. This section documents them.

- The **[full alphabetical index](generated/index-table.md)** and
  category pages are generated directly from BlueSky's command registry
  every time these docs are built, so they always match the code.
- The pages below are hand-written deep dives for some of the most commonly
  used commands, with worked examples.
